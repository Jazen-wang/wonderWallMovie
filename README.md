# 项目名称：vue-movie

> 一个购票网站平台 -- By sysu团队

## 前端开发

``` bash
cd app-vue

# 安装依赖
npm install

# 自动监听 localhost:8080
npm run dev

# 正式环境部署
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
